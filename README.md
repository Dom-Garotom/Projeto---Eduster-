## Descrição do Projeto

A **Eduster** é uma plataforma criada para facilitar o **gerenciamento e agendamento de salas** em uma instituição. Este projeto faz parte de uma disciplina de **Web Design** e tem como objetivo fornecer experiência prática sobre as etapas de desenvolvimento de um sistema real, desde o planejamento até a implementação.

O sistema foi desenvolvido utilizando **JavaScript**, **HTML** e **CSS**, com foco em design responsivo e uma interface intuitiva para garantir que usuários possam navegar facilmente pelo sistema. Durante o desenvolvimento, também foram aplicados conceitos e boas práticas de **desenvolvimento web** aprendidos em sala de aula.

## Funcionalidades Principais

- **Agendamento de Salas**: Usuários podem reservar salas de forma rápida e prática, através de uma interface simples e acessível.
- **Confirmação de Agendamentos**: Modais são utilizados para confirmar os agendamentos e fornecer feedback ao usuário.
- **Listagem de Salas**: O sistema permite visualizar quais salas estão ocupadas ou livres, facilitando a gestão e uso eficiente dos espaços.
- **Autenticação de Usuários**: Futuramente, o sistema contará com um módulo de login e registro, garantindo acesso seguro às áreas restritas da plataforma.

## Tecnologias Utilizadas

- **HTML**: Estrutura das páginas e formulários.
- **CSS**: Estilização, incluindo design responsivo e otimizações para diferentes dispositivos.
- **JavaScript**: Lógica de agendamento, interações com a interface e manipulação de dados.

## Desafios e Aprendizados

Durante o desenvolvimento deste projeto, vários desafios surgiram, especialmente em relação à criação de uma interface totalmente responsiva. Aprender a manipular **modais dinâmicos** e garantir uma experiência de usuário fluida, independentemente do dispositivo, foi uma parte essencial do processo.

Outro ponto importante foi o planejamento e implementação da **estrutura de autenticação** que, embora ainda esteja em progresso, tem ensinado muito sobre como proteger sistemas e gerenciar usuários.
